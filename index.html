Research

  1. Hybridizing Reinforcement Learning and Metaheuristics to design robust optimization framework

    Undergraduate thesis work
    Supervisor: Dr. Muhammad Ali Nayeem, Assistant Professor, CSE, BUET, Dhaka
    Keywords: Neural Combinatorial Optimization , Reinforcement Learning(RL), Graph Neural Network (GNN)

    We aim to solve computer optimization problems such as DVRP(Dynamic Vehicle Routing Problem), TSP(Travelling Salesman Problem) using deep reinforcement learning and different meta-heuristic techniques such as ant colony, evolutionary algorithm etc. We leverage GNN to get graph embedding from the problem instance. Using that embedding we generate heuristic which is then used in the solution.


  1. Novel Class Discovery

    Remote Part Time Research Assistant
    Supervisor: Dr. Avijoy Chakma, Assistant Professor, CS, Bowie State University, Maryland, USA
    Keywords: Computer Vision, Novel Class, Clustering, Open-world learning

    We aim to discover new semantic classes in an unlabelled datset by leveraging the knowledge acquired from another labeled data set where it is assumed that the classes in the labeled and unlabeled set are disjoint. To this end, we use various self-supervised learning(SSL) techniques to learn a better representaton of the samples then apply various clustering algorithm.
